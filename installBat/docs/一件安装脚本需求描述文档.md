# opencode插件一件安装脚本需求说明

---

## 一、需求说明

### 前提条件
默认用户已安装好opencode

### 需求内容
帮我生成一个一件安装的bat脚本文件，在windows系统内双击即可运行，满足以下功能：
1、将此脚本所在同文件夹内的`message-bridge.plugin.js`文件复制到`C:\Users\USERPROFILE\.config\opencode\plugins`文件夹内
  若plugins文件夹不存在，则先创建该文件夹;
  若plugins文件夹内已存在`message-bridge.plugin.js`文件，则直接替换;
  说明：
  `C:\Users\USERPROFILE\.config\opencode\plugins`指的是用户的opencode插件配置路径，如我自己电脑的路径为`C:\Users\SCY\.config\opencode\plugins`
2、读取`C:\Users\USERPROFILE\.config\opencode`内的`message-bridge.json`文件，根据用户输入的ak,sk的值替换其中对应字段的值;
  若`message-bridge.json`文件不存在则新建；
  若`message-bridge.json`文件已存在，则复用该文件；
  若`message-bridge.json`中已存在ak，sk字段，则将用户输入的ak，sk值覆盖替换；
  bat脚本输入ak，sk值支持用户手动输入，也支持快捷键粘贴；
  输入ak，sk时要有良好的交互体验；

  `message-bridge.json`文件内容参考
  ```json
  {
    "xxx": "test",
    "auth": {
      "ak": "Ivk1/NQO5",
      "sk": "ME4CA454ASFAF"
    }
  }
  ```
